# Autodoprava Kopeček

Bilingual marketing and lead-generation website for a family bus and freight
transport company in Hodonín (autobusyhodonin.cz). Focus: trust, services, fleet,
and easy contact.

## Tech stack

- **Framework:** Next.js (App Router, Turbopack)
- **Language:** TypeScript
- **UI:** React, Tailwind CSS, Lucide icons
- **Rendering:** Statically rendered marketing pages, no database

## Connected third parties

- **Google Maps** — embedded map showing the company location on the contact page.

## Key libraries

- `tailwind-scrollbar-hide` — hides scrollbars on horizontal carousels.
- `lucide-react` — icon set.
